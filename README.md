# 🚀 AdFlow Pro

AdFlow Pro is a **sponsored ads marketplace platform** where users can create listings, moderators review them, and admins approve them before publishing.

---

## 📌 Project Overview

This project is built as part of an **Advanced MERN / Full Stack Development** course.
It simulates a real-world workflow system with multiple roles and controlled ad publishing.

---

## ✨ Features

* 👤 User Authentication (Client, Moderator, Admin)
* 📝 Ad Creation & Management
* 🔍 Moderation System (Review & Reject Ads)
* 💳 Payment Submission & Verification
* 📢 Ad Publishing & Expiry System
* 📊 Dashboard for different roles

---

## 🛠️ Tech Stack

* **Frontend:** Next.js (React)
* **Backend:** Node.js + Express (Planned)
* **Database:** Supabase / MongoDB
* **Deployment:** Vercel

## 🚀 Getting Started (Local Setup)

### 1️⃣ Clone the repository

```
git clone https://github.com/sanafa23-bcs026/adflow-pro.git
```

### 2️⃣ Go to client folder

```
cd adflow-pro/client
```

### 3️⃣ Install dependencies

```
npm install
```

### 4️⃣ Run the project

```
npm run dev
```

### 5️⃣ Open in browser

```
http://localhost:3000
```
